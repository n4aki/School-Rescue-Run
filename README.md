# School Rescue Run

ブラウザで遊べる、依存なしのシンプルな 2D 横スクロールアクションゲームです。`canvas` を使って描画しています。

## ローカル起動

`index.html` を直接ブラウザで開くか、必要に応じて簡易サーバーを起動してください。

```powershell
cd <project-folder>
python -m http.server 8000
```

サーバーを起動した場合は、ブラウザで `http://localhost:8000` を開きます。

## 操作

- 左右移動: `←` / `→` または `A` / `D`
- ジャンプ: `Space` / `W` / `↑`
- リスタート: `R`

## ゲーム内容

- 足場を渡って右端のゴールを目指します。
- 穴に落ちるか、敵やトゲに当たると残機が 1 減ってスタート地点に戻ります。
- コインを集めるとスコアとしてカウントされます。
- 残機が 0 になるとゲームオーバーです。`R` で最初からやり直せます。

## ファイル構成

- `index.html`: エントリーポイント
- `style.css`: レイアウトと見た目
- `game.js`: ゲームループ、入力、当たり判定、描画
- `.nojekyll`: 補助ファイル

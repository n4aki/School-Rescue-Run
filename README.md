# School Rescue Run

4月に入学したばかりの新入生が校内を駆け回り、各ステージで捕まっている部活動の生徒を救出しながら、最後に校長室でボスを倒して学校を救う 2D 横スクロールアクションゲームです。  
HTML / CSS / JavaScript だけで動く、依存なしの軽量プロトタイプです。

## ローカル起動

`index.html` を直接ブラウザで開くか、必要に応じて簡易サーバーを起動してください。

```powershell
cd <project-folder>
python -m http.server 8000
```

サーバーを起動した場合は、ブラウザで `http://localhost:8000` を開きます。

## ゲームの流れ

1. 新入生として教室からスタートします。
2. 各ステージで捕まっている仲間を救出します。
3. 救出した仲間は次のステージ以降でプレイアブルとして使えるようになります。
4. 最後に校長室で校長とのボス戦に挑みます。

実装済みステージ:

- 教室
- 理科室
- 体育館
- 校庭
- 校長室

## 操作

- 左右移動: `←` / `→` または `A` / `D`
- ジャンプ: `Space` / `W` / `↑`
- キャラ切替: `C`
- 攻撃: `F`
- リスタート: `R`

### スマホ操作

- 画面下のタッチボタンで `左移動` `右移動` `Jump` `Attack` `Switch` を操作できます
- 左右移動ボタンは長押しで移動を継続できます
- 縦画面でも遊べますが、横向きプレイ推奨です

## プレイアブルキャラ

- 新入生: 標準性能の主人公
- 美術部: 隠し足場を見つけて渡れる
- 科学部: スイッチ扉を解除できる
- バスケ部: ジャンプが高い
- 陸上部: 移動速度が速い

## プロトタイプ仕様

- 救出した仲間はHUDに反映され、`C` で切り替えられます
- 通常ステージは「救出対象に接触してから出口へ到達」でクリアです
- 校長室では短距離攻撃でボスHPを削るボス戦があります
- 敵、トゲ、薬品、ボス攻撃に当たると残機が減ります
- PC のキーボード操作とスマホのタッチ操作は共存します

## ファイル構成

- `index.html`: HUD、キャンバス、操作説明
- `style.css`: レイアウトとHUDの見た目
- `game.js`: ステージ定義、キャラ能力、ゲームループ、描画、ボス戦
- `.nojekyll`: 補助ファイル
